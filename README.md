# CoachMe

CoachMe is an AI-based fitness and nutrition assistant that generates personalized workout and diet recommendations.

## Project Description

This project builds a chatbot that helps users with:
- Workout planning
- Nutrition advice
- Fitness guidance

The system is trained using two datasets:
- Fitness dataset (workouts, exercises, training plans)
- Nutrition dataset (meals, calories, macronutrients)
